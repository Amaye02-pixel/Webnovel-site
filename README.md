# 📚 NovelLuxe – Web Novel Reading Platform (School Project)

## Overview

**NovelLuxe** is a simple web novel reading platform built using **HTML, CSS, and JavaScript**.
This project was created as a **practice project for learning web development** and demonstrating how a basic online novel platform works.

The website allows users to browse novels, open chapters, and read stories in a clean reading interface.

This project focuses on:

* Front-end web development
* UI/UX layout design
* Interactive JavaScript features
* Responsive web design

## 🚀 Features

### 📖 Reading Platform

* Dynamic **novel and chapter system**
* Clean **reader interface**
* Automatically generated **chapters and content**

### 🎬 UI Features

* Cinematic animated homepage
* Netflix-style horizontal novel shelves
* 3D interactive book cards
* Dark / Light theme toggle

### 🔎 Interactive Functions

* Novel **search system**
* **Rating system** (1–5 stars)
* **Comment system** using browser storage

### 📱 Responsive Design

* Mobile-friendly layout
* Adaptive UI for tablets and phones

---

## 🛠 Technologies Used
